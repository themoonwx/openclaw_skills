# CC任务派发Skill

## 概述

派发任务给Claude Code (CC) 子Agent

## 派发格式

```
---

**📋 任务名称**：xxx

**任务简介**：xxx

**⏱️ 预计时间**：X分钟
**⌛ 超时**：X分钟
**📊 复杂程度**：简单任务 / 复杂任务 / 巨量工程

---

### 任务内容
...
```

## CC完成任务后的返回格式

要求CC完成后按以下格式返回：

```
## ✅/❌ 任务完成

**任务名称**：xxx
**状态**：成功/失败
**运行时长**：X分钟

### 执行摘要
xxx

### 修改的文件
- file1.ts
- file2.md

### 错误详情（失败时）
xxx

### 后续建议
1. 下一步做什么
2. 需要注意什么
```

## 使用方法

使用 `sessions_spawn` 工具派发任务

### 参数

| 参数 | 说明 |
|------|------|
| agentId | 固定为 "main" |
| task | 任务内容（包含上面格式） |
| label | 任务标签 |
| runTimeoutSeconds | 超时时间（秒） |

### 示例

```javascript
{
  "agentId": "main",
  "task": "## 任务：xxx\n\n### 预计时间：5分钟\n### 超时：10分钟\n\n任务内容...",
  "label": "cc-xxx",
  "runTimeoutSeconds": 600
}
```

## 复杂程度判断

- **简单任务**：5分钟内完成
- **复杂任务**：10-30分钟
- **巨量工程**：30分钟以上
